# Голуби ходят буквой Г

Голубь — вестник мира, пока не доходит до дележки частной собственности.

Идею про голубей, играющих в шахматы, увёл из-под носа двух спорщиков. Шефа из «Братьев Пилотов» стащил из аватарки одного из участников. Все права на голубей, шахматы, слова, предложения, абзацы и параграфы принадлежат тому, кому принадлежат.
